# 霊視鑑定士 初回DM返信テンプレ集

無料鑑定の申し込みが来てから、有料鑑定・リピートにつなぐまでの一連のDMテンプレです。
そのままコピーして、`〔　〕` を埋めて使ってください。

## 使う前に（3つの鉄則）
- **速さが命**：申し込みが来たら、できるだけ早く返す。早さそのものが信頼になる。
- **名前を必ず入れる**：`〔お名前〕さん` を入れるだけでテンプレ感が消え、「私を見てくれている」に変わる。
- **出し惜しみしない**：無料でも全力で答える。ここでの誠実さが、有料リピートの土台になる。

---

## テンプレ①　受付（申し込みが来たら即レス）

```
〔お名前〕さん、はじめまして😊
メッセージありがとうな。魂を救う霊視鑑定士です🔮

今から〔お名前〕さんのこと、しっかり視させてもらうね。
まず、いちばん心にひっかかってること・お悩みを、そのままの言葉で聞かせてもらえる？💌

恋・仕事・人間関係・お金…どんな内容でも大丈夫。
一人で抱え込まんでええからな🕊️
```

## テンプレ②　ヒアリング受け（お悩みが届いたら）

```
〔お名前〕さん、話してくれてありがとう。
〔お悩みを一言でやさしく言い換える〕…そら、しんどかったな。よう頑張ってきたんやね。

今からカードとつなげて、〔お名前〕さんのこと、じっくり視ていくね。
少しだけ待っててな🔮
```

> この間に：ダッシュボードでカードを引く → コピー → Claudeの鑑定プロジェクトで鑑定文を作成 → Canvaで「鑑定書」に仕上げる。

---

## テンプレ③　鑑定結果を送る（鑑定書＋添え文）★最重要

`〔鑑定書の画像〕` を添付し、下の文を添えて送ります。最後の「次なる課題」がリピートの肝です。

```
〔お名前〕さん、お待たせ🌙 鑑定書ができたよ。
〔鑑定書の画像を添付〕

〔鑑定のいちばん大事なポイントを1〜2行で〕
今回のお悩みは、この流れで動いていけば、きっと大丈夫やで。

──ただな。視ていく中で、〔お名前〕さんの根っこに、もう一つ繰り返しやすいテーマが視えたんよ。
そこを一緒にほどいていけたら、同じことでしんどくなる流れも、きっと変わっていくはず。

もし気になったら、そこまでじっくり視る鑑定で、一緒に見ていこか😊
```

**設計のポイント**：`今回の解決策` ＋ `まだ気づいていない根っこの課題` をセットで渡す。
「今の悩みは解決したけど、根本もこの人に見てほしい」という気持ちが自然に生まれ、有料鑑定につながる。

```
じっくり視る鑑定はこちら👇
・〔有料メニュー名〕：〔料金〕（〔ボリューム・所要時間など〕）

〔お名前〕さんが特に深めたいのはどのあたりか、教えてくれる？
そこに合わせて、根っこまで一緒に視ていくね🔮
```

## 個別対応のコツ

- お悩みの中の言葉を1つ拾って返す（例：「“既読スルー”がずっと気になってるんやね」）。これだけで一気に「わかってくれてる」に変わる。
- 質問には漏れなく答える。1つでも流すと信頼が下がる。
- 鑑定書は毎回同じ世界観（プラム×ゴールド）で。ブランドとして覚えてもらえる。

## 守ること（信頼と安全）

- 「必ず復縁する」などの断定的な保証はしない。
- 深刻な悩み（強い落ち込み・自分を傷つけたい気持ち・生活が立ち行かない等）が見えたら、占いは寄り添いにとどめ、専門家や信頼できる人・相談窓口を頼るようやさしく添える。
- 不安を過度に煽って契約させない。あくまで「気づき」と「希望」で next につなぐ。
- 実績や経歴を誇張しない。
